Instrucciones:
  • Selecciona entre acordes normales o especiales pinchando el botón correspondiente.
  • Para acordes normales, selecciona el Tono y luego el Modo (mayor, menor, etc.).
  • Para acordes especiales, selecciona directamente el acorde.
  • Cada acorde puede tener más de una Versión o Posición.
  • Para ver las diferentes versiones de un acorde, pincha el botón Siguiente.
  • Utiliza el botón Notación para cambiar entre las notaciones latina e inglesa.
